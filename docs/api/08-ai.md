# 1. Get Recommended Businesses

Retrieve a list of recommended business branches nearby tailored to the user.

*   **URL**: `/recommendations/businesses`
*   **Method**: `GET`
*   **Access**: Public (Authenticated users receive personalized results; guests receive location-based averages).

### Query Parameters
*   `lat`: Current user latitude (mandatory).
*   `lon`: Current user longitude (mandatory).
*   `limit`: Number of items to return (default 5).

### Response (200 OK)
```json
{
  "success": true,
  "message": "Recommended businesses retrieved successfully",
  "data": [
    {
      "branchId": 1,
      "businessName": "Zen Luxury Spa",
      "branchName": "Zen Spa District 1",
      "distanceKm": 1.25,
      "averageRating": 4.9,
      "popularityScore": 0.88,
      "recommendationScore": 0.941
    }
  ],
  "errors": null,
  "timestamp": "2026-07-17T11:29:38Z"
}
```

# 2. Get Recommended Services

Get popular or recommended services.

*   **URL**: `/recommendations/services`
*   **Method**: `GET`
*   **Access**: Public

### Query Parameters
*   `branchId`: Filter recommendations to a specific branch (optional).
*   `limit`: Max elements (default 5).

---

# 3. Get Recommended Employees

Suggest staff members based on rating and matching service popularity.

*   **URL**: `/recommendations/employees`
*   **Method**: `GET`
*   **Access**: Protected (`CUSTOMER` only)

### Query Parameters
*   `branchId`: The target branch (mandatory).
*   `serviceId`: The service being booked (optional).
